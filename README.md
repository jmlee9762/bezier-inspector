# Bézier Inspector

![Overview](overview.png)

A CEP extension for Adobe Illustrator that visualizes anchor points, Bézier handles, connector lines, and font metrics directly on the artboard.

## Features

- Anchor point visualization
- Bézier handle visualization
- Handle connector lines
- Baseline, x-height, Cap, Ascent, Descent
- Adjustable colors, sizes, and metric captions
- Editable overlay groups

## Installation

Download the latest release.

Copy the folder:

com.ju.bezierinspector

to:

macOS

~/Library/Application Support/Adobe/CEP/extensions/

Windows

%APPDATA%/Adobe/CEP/extensions/

Restart Illustrator.

Window → Extensions → Bézier Inspector
